Le site internet utilise SQL SERVER

Il faut installer les drivers sql server sur xampp.
télécharger les drivers : https://docs.microsoft.com/fr-fr/sql/connect/php/loading-the-php-sql-driver?view=sql-server-ver15
Tuto : https://www.youtube.com/watch?v=tqMZoR-Q-NY

Utiliser le script.sql pour créer la base de données dans SQL SERVER

Utiliser donnees.sql pour avoir les données de base du site.

Connectez-vous en tant qu'administrateur (login : admin et mdp : admin) dans la page de connexion pour gérer la base de données (création de client, création des semaines, création des chalets...).

Enfin, connectez-vous en tant que client pour tester les résultats.
